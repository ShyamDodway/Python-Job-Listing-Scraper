#  Python Job Listing Scraper for Data Analysis and Market Trends

##  Project Overview
This project scrapes job listings from **AmbitionBox** and **Glassdoor** using `Python`, `Requests`, and `BeautifulSoup`.  
It builds a structured dataset of job postings to analyze **in-demand skills**, **salary trends**, and **industry requirements** in the Data Science domain.

---

##  Features
- Scrapes 1,000+ job postings from AmbitionBox and Glassdoor.
- Extracts key job details:
  - Job Title  
  - Company Name  
  - Location  
  - Experience Required  
  - Salary (if available)  
  - Skills / Tags
- Exports results into a clean CSV file.
- Code is modular and easy to extend for other job roles.

---

##  Technologies Used
- **Python 3**
- **BeautifulSoup4**
- **Requests**
- **Pandas**
